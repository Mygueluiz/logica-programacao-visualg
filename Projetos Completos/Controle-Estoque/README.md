# Controle de Estoque

Projeto desenvolvido em Visualg para praticar lógica de programação.

## Objetivo

Criar um sistema simples de controle de estoque, permitindo cadastrar produtos, calcular valores e controlar quantidades.

## Tecnologias utilizadas

- Visualg
- Lógica de Programação

## Execução

Programa desenvolvido e testado no Visualg 3.0.7.
